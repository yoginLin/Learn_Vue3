### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
如果把应用的所有逻辑都放在一个组件中，那么这个组件会十分的臃肿和不好维护
组件化的核心思想是对组件进行拆分，拆分成一个个小的组件
再把这些组件组合嵌套在一起，最终形成我们的应用程序
